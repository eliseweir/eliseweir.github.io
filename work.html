Programs

  • HyperbolicTriangle

    • For the non-expert: produces nxn matrices a, b, c that satisfy the equations a^p = b^q = c^r = abc = +/- I and no others!

    For the expert: produces an nxn faithful, irreducible representation for hyperbolic triangle groups

    T(p,q,r) = <a,b,c|a^p = b^q = c^r = abc = 1>

    in PSL(n,R) by composing the holonomy representation T(p,q,r) -> PSL(2,R) with the unique irreducible representation PSL(2,R) -> PSL(n,R).

    Run it from a SageMath shell, or edit the values of p, q, r, and n below to try it out in your browser!
